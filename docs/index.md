# Welcome to Boilerplate SaaS

This is a production-ready SaaS template built with **FastAPI** (Python) and **Next.js** (TypeScript).

## Features

- **Multi-tenancy:** Organizations and memberships built-in.
- **Authentication:** JWT-based auth with email verification and Google OAuth.
- **Admin Tools:** Global dashboard, system stats, and user impersonation.
- **File Management:** S3-compatible storage (MinIO) integration.
- **Developer Experience:** Multi-step onboarding, automated migrations, and structured logging.
- **Deployment:** Zero-downtime deployments with Kamal.
